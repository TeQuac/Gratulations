# Gratulations

Mobile-first Geburtstagskalender mit Monats-/Wochenansicht, Swipe-Navigation und lokaler KI-Textgenerierung für personalisierte Geburtstagswünsche.

## Benachrichtigungen

- Benachrichtigungen aktivieren.
- Gewünschte Uhrzeit auswählen.
- Die App erinnert dann jeden Tag automatisch an Geburtstage.
- Im Gratulationsfenster kannst du Wünsche direkt ohne Backend über **E-Mail** (`mailto`) oder **WhatsApp** (`wa.me`) mit vorausgefülltem Text öffnen.
